# 🇬🇧 영어 학습 에이전트

Guardian 뉴스 기사로 영어 표현을 배우는 LangGraph 기반 학습 봇입니다.

## 프로젝트

영어 표현을 입력하면:

1. Guardian에서 관련 기사 자동 검색
2. 기사 번역 + 표현 분석 + 예문 생성 (동시에!)
3. 직접 문장 써보고 AI 피드백 받기
4. 4지선다 퀴즈 → 틀리면 자동 재도전
5. 학습 기록 자동 저장

## 주요 기능

1. Tools : Guardian 검색 / 난이도 판별 / 유의어 찾기
2. 병렬 실행 : 번역·분석·예문을 동시에 생성 (Send API)
3. 조건 분기 : 퀴즈 오답 시 자동으로 재도전 (Conditional Edge)
4. 메모리 : 세션 끊겨도 학습 기록 유지 (MemorySaver)

## 실행 흐름

```
표현 입력
↓
Guardian 기사 검색
↓
번역 / 분석 / 예문 (동시 생성)
↓
쓰기 연습 + AI 피드백
↓
퀴즈 → 오답이면 재도전
↓
학습 기록 저장
```
